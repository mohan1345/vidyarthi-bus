# Design Screens

The `designs/` folder contains HTML prototypes for the planned Vidyarthi Bus experience.

## Screens

- `splashscreen.html` - app opening screen.
- `homedashabord.html` - main student dashboard concept.
- `route-selection.html` - route search and selection concept.
- `live-crowd.html` - live crowd status concept.
- `report-status.html` - status reporting flow.
- `report-histroy.html` - previous report history concept.
- `running-offline.html` - offline running mode concept.
- `shared-auto contacts.html` - shared auto contact concept.
- `settings.html` - app settings concept.

## Notes

Some prototype filenames preserve their original draft names. They can be renamed later if the app links to them directly or if the design folder is published as static documentation.
